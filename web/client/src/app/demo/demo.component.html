OnBoardPi

The open-source web based OBD2 application for vehicles.

  View on GitHub

Features

Dashboard

Watch diagnostic data on a convienient dashboard with gauges, curves, or numeric values.

Diagnostics

View stored, pending, or freeze DTCs (diagnostic trouble codes i.e. check engine light) and current status.

Data Streams

Visualize any OBD2 values on live charts, realtime system information, or all OBD2 values in a continuous stream.
